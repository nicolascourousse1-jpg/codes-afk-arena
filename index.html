Comment activer (redeem) un code AFK Arena ?

La première chose à savoir pour profiter de ces récompenses régulières est comment activer un code de remboursement dans AFK Arena.

Pour les vieux loups de mer, attention ! Depuis décembre, le processus a un peu changé et il faut bousculer vos vieilles habitudes. Dorénavant, il faut se rendre sur le site officiel via un navigateur internet (PC ou mobile), en plus d'avoir votre jeu ouvert à côté. La manipulation est un peu plus contraignante mais reste très simple.

Temps nécessaire : 1 minute.

Étapes pour activer un code sur AFK Arena :

  1. Ouvrir un navigateur et se rendre sur le site officiel. Le plus pratique pour éviter les allers-retours, c'est de l'ouvrir sur votre PC.
  2. Rentrer votre UID et demander le code de vérification. Il se trouve dans votre jeu en cliquant sur votre avatar puis en ouvrant vos détails. C'est une suite de chiffres.
  3. Rentrer le code de vérification reçu. Ce dernier est envoyé dans vos messages du jeu AFK Arena. Ouvrez donc votre jeu et vérifiez vos mails in-game.
  4. Récupérer votre cadeau. Une fois le code vérifié, votre récompense gratuite se trouve dans vos messages in-game : cliquez sur l'icône de la messagerie pour la récupérer !

Pourquoi utiliser les codes cadeaux AFK Arena ?

La plupart des joueurs utilisent ces codes gratuits pour récupérer un peu de ressources et faire monter leur compte petit à petit. Chaque code offre des récompenses différentes (diamants, or, pierres d'âme rares et élites, et parfois un héros lors d'événements spéciaux) et possède sa propre date d'expiration.

  • Ils donnent un vrai coup de pouce en début de partie pour invoquer plus de héros.
  • Ils sont 100 % gratuits et utilisables sur Android comme sur iOS.
  • Ils tombent souvent lors des mises à jour, événements ou grandes fêtes : surveillez les annonces !

Quelles récompenses rapportent les codes AFK Arena ?

Chaque code donne des récompenses différentes, mais toutes servent à renforcer plus vite votre compte sans dépenser. Voici les ressources que vous croiserez le plus souvent :

  • Diamants (1 000 à 3 000) : la monnaie universelle, pour les invocations, les emplacements de sac et les tickets de Labyrinthe.
  • Parchemins d'invocation : un lot de 10 peut sortir un compte débutant de l'early game en un éclair.
  • Or & Essence de héros : indispensables pour l'équipement et l'ascension de vos héros une fois les niveaux élevés.
  • Coffres de héros au choix : ils évitent le hasard et vous laissent choisir un carry de premier plan.
  • Pierres d'âme, cartes d'observation des étoiles, emblèmes du temps : pour monter en puissance vos meilleurs héros.

D'où viennent les codes AFK Arena et où les trouver ?

Les codes AFK Arena sont des CD-Keys officiels publiés par Lilith Games (ou des marques partenaires). Comme le jeu n'impose aucune pub forcée, ces codes sont la « monnaie de bonne volonté » du studio. Ils apparaissent surtout :

  • lors des mises à jour et patchs du jeu ;
  • pendant les événements mondiaux (anniversaires, Noël, Nouvel An lunaire) ;
  • à l'occasion des crossovers (Re:ZERO, The Witcher, Persona 5…) ;
  • via les réseaux sociaux officiels (paliers de likes/retweets, lives).

Le plus simple reste de garder cette page en favori : nous la mettons à jour dès qu'un nouveau code tombe, pour vous éviter de fouiller le Discord et les réseaux vous-même.

Codes AFK Arena Companions et Classic : attention au bon serveur

AFK Arena possède deux types de serveurs, Classic et Companions, et chacun a sa propre liste de codes. Un code AFK Arena Companions ne fonctionnera pas sur le serveur Classic, et inversement. Avant de saisir un code sur le site de remboursement, pensez donc à sélectionner le bon serveur (Classic ou Companions), sous peine d'un message d'erreur. Dans nos tableaux ci-dessus, les codes sont valables sur le serveur Classic sauf mention contraire.

Foire aux questions (FAQ)

Comment activer un code sur AFK Arena ?

Rendez-vous sur le site officiel de remboursement via un navigateur, entrez votre UID, demandez puis saisissez le code de vérification reçu dans vos messages in-game, validez, puis entrez un code actif de notre liste. La récompense arrive dans votre boîte mail in-game.

Où trouver mon UID AFK Arena ?

Touchez votre avatar en haut à gauche de l'écran pour ouvrir les détails du compte : votre UID y est affiché sous forme d'une suite de chiffres. C'est cet identifiant qu'il faut saisir sur le site de remboursement.

Les codes fonctionnent-ils sur le serveur de test ?

La plupart du temps, oui : un même code marche sur le serveur officiel et sur le serveur de test. Il existe seulement de rares exceptions où un code ne fonctionne pas sur le serveur de test.

À quelle fréquence de nouveaux codes sortent-ils ?

Il n'y a pas de calendrier fixe. Les développeurs offrent généralement des codes lors d'une nouvelle mise à jour, à l'approche d'un événement spécial ou d'une grande fête. Il se passe rarement plus de deux semaines entre deux annonces.

Existe-t-il un code de triche AFK Arena ?

Non, il n'existe aucun code de triche AFK Arena : pas de hack de diamants, d'or ou de puissance. Tout ce qui le promet est une arnaque. Le seul moyen sûr d'obtenir des ressources gratuites est d'utiliser les codes de remboursement officiels listés sur cette page.

Le code « 1000 free summons » AFK Arena existe-t-il ?

Non. Aucun code officiel ne donne 1000 invocations gratuites sur AFK Arena : ces offres sont fausses et cherchent souvent à voler vos identifiants. Les vrais codes rapportent surtout des diamants, des parchemins et des ressources, comme dans nos tableaux ci-dessus.

Un code AFK Arena ne marche pas, pourquoi ?

Le plus souvent, le code est expiré ou a déjà été utilisé sur votre compte (chaque code n'est valable qu'une fois). Vérifiez aussi l'orthographe exacte (majuscules comprises), le bon serveur (Classic ou Companions) et que le code de vérification UID n'a pas dépassé son délai de 2 minutes.

Qu'est-ce qu'AFK Arena ?

AFK Arena est un RPG mobile de Lilith Games basé sur la collection de héros et la progression « idle » (votre équipe farme même hors ligne). On y constitue des équipes de différentes factions pour avancer dans la campagne, affronter des boss et grimper en PvP. Les codes accélèrent le démarrage en offrant des ressources d'invocation et d'amélioration.
